## README – Age Classification Script

### Description

This script classifies a user based on their age and displays a personalized message.
It validates numeric input and determines the life stage of the user.

### Functionality

* Receives the user’s name and age.
* Classifies the age into predefined ranges (child, minor, adult, mature adult).
* Displays messages using `console.log` and `alert`.
* Handles invalid or unrealistic age inputs.

### Purpose

Practice conditional logic (`if / else if / else`) and basic input validation in JavaScript.
